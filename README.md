# Chamador de Senhas Simples

Chamador de senhas simples com armazenamento local desenvolvido com Node.js (Javascript).

## Instalação

Requer o [Node.js](https://nodejs.org/) v10+.

Instale as dependências do projeto

```sh
cd chamador-senhas
npm i
npm start
```
